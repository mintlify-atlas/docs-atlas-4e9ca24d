## Overview

The `ModelController` handles all document CRUD operations including creating, reading, updating, deleting documents, managing tags, assignments, and searching.

## Document Creation

### newDoc()

Creates a new document with default properties and adds it to the local cache.

```typescript
const doc = await core.model.newDoc({
  doctype: 'Sales Invoice'
});
```

<ParamField path="newDocParams" type="NewDocParams" required>
  Document creation parameters
  <Expandable title="NewDocParams properties">
    <ParamField path="doctype" type="string" required>
      The doctype to create
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="return" type="RenovationDocument">
  New document with default fields:
  - `docstatus`: 0
  - `__islocal`: 1
  - `__unsaved`: 1
</ResponseField>

### copyDoc()

  <Expandable title="GetDocParams properties">

      Target doctype

### getList()

Retrieves a list of documents with filtering and pagination.

```typescript
const response = await core.model.getList({
  doctype: 'Sales Invoice',
  fields: ['name', 'customer', 'total'],
  filters: { status: 'Draft' },
  orderBy: 'creation desc',
  limitPageStart: 0,
  limitPageLength: 20
});
```

<ParamField path="getListParams" type="GetListParams" required>
  List retrieval parameters
  <Expandable title="GetListParams properties">
    <ParamField path="doctype" type="string" required>
      Target doctype
    </ParamField>
    <ParamField path="fields" type="string[]">
      Fields to include. Default: `['name']`. Use `['*']` for all fields
    </ParamField>
    <ParamField path="filters" type="DBFilter">
      Filter conditions
    </ParamField>
    <ParamField path="orderBy" type="string">
      Sort order. Default: 'modified desc'
    </ParamField>
    <ParamField path="limitPageStart" type="number">
      Pagination offset. Default: 0
    </ParamField>
    <ParamField path="limitPageLength" type="number">
      Number of records. Default: 99
    </ParamField>
    <ParamField path="parent" type="string">
      Parent doctype for child tables
    </ParamField>
    <ParamField path="tableFields" type="object">
      Child table fields to include `{ 'items': ['item_code', 'qty'] }`
    </ParamField>
    <ParamField path="withLinkFields" type="string[]">
      Link fields to populate
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="data" type="Array<object | RenovationDocument>">
  List of documents. Returns `RenovationDocument[]` if `fields: ['*']`
</ResponseField>

<Note>
The `tableFields` and `withLinkFields` parameters require the `renovation_core` app.
</Note>

## Code Examples

### Complete CRUD Flow

<CodeGroup>
```typescript Create and Save
// Create new document
const invoice = await core.model.newDoc({
  doctype: 'Sales Invoice'
});

// Set values
invoice.customer = 'CUST-001';
invoice.posting_date = '2024-01-15';

// Add child items
const item = await core.model.addChildDoc({
  doc: invoice,
  field: 'items'
});
item.item_code = 'ITEM-001';
item.qty = 10;

// Save
const saveResponse = await core.model.saveDoc({
  doc: invoice
});

if (saveResponse.success) {
  console.log('Saved:', saveResponse.data.name);
}
```

```typescript Read and Update
// Get document
const response = await core.model.getDoc({
  doctype: 'Sales Invoice',
  docname: 'SI-001'
});

if (response.success) {
  const invoice = response.data;
  
  // Update single field
  const updateResponse = await core.model.setValue({
    doctype: 'Sales Invoice',
    docname: 'SI-001',
    docfield: 'status',
    value: 'Paid'
  });
}
```

```typescript Submit and Cancel
// Submit document
const submitResponse = await core.model.submitDoc({
  doc: invoice
});

if (submitResponse.success) {
  console.log('Submitted with docstatus:', submitResponse.data.docstatus);
  
  // Later: cancel if needed
  const cancelResponse = await core.model.cancelDoc({
    doc: invoice
  });
}
```
</CodeGroup>

### List with Filters

```typescript
const response = await core.model.getList({
  doctype: 'Sales Invoice',
  fields: ['name', 'customer', 'total', 'status'],
  filters: {

    posting_date: ['between', ['2024-01-01', '2024-12-31']]
  },
  orderBy: 'total desc',
  limitPageStart: 0,
  limitPageLength: 50
});

console.log(`Found ${response.data.length} invoices`);
```
