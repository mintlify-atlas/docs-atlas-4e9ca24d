# Welcome to Renovation Core

Renovation Core is a powerful TypeScript SDK that provides a complete solution for interacting with Frappe backend systems. Built with modern development practices, it offers type-safe APIs, real-time capabilities, and comprehensive document handling.

## What is Renovation Core?

Renovation Core is the missing JavaScript/TypeScript SDK for Frappe backends. It abstracts the complexity of Frappe's API layer and provides an intuitive, promise-based interface for building front-end applications. Whether you're building a web app, mobile app, or desktop application, Renovation Core gives you the tools you need to interact seamlessly with your Frappe backend.

## Core Capabilities

### Type-Safe APIs

Renovation Core is written in TypeScript, providing full type safety and IntelliSense support in modern IDEs. You'll catch errors at compile time and enjoy autocomplete for all API methods.

### Promise-Based

All async operations return promises, making it easy to work with modern async/await syntax and integrate with your existing JavaScript/TypeScript codebase.

### Modular Controllers

The SDK is organized into focused controllers:
- **AuthController**: Handle authentication and user sessions
- **ModelController**: Manage document CRUD operations
- **MetaController**: Access doctype metadata
- **PermissionController**: Check and manage permissions
- **StorageController**: Upload and manage files
- **TranslationController**: Handle internationalization
- **DashboardController**: Manage dashboards and UI components

### Local Caching

Renovation Core intelligently caches documents and metadata locally, reducing network requests and improving application performance.
