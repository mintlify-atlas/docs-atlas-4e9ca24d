# Quick Start

This guide will walk you through the essential steps to start using Renovation Core in your application.

## Initialize Renovation Core

<Steps>
  <Step title="Import and create instance">
    Start by importing the Renovation class and creating a new instance:
    
    ```typescript
    import { Renovation } from '@leam-tech/renovation-core';
    
    const renovation = new Renovation();
    ```
  </Step>
  
  <Step title="Initialize with configuration">
    Initialize Renovation Core with your Frappe backend URL:
    
    ```typescript
    await renovation.init({
      hostURL: 'https://your-frappe-site.com',
      backend: 'frappe'
    });
    ```
    
    <Note>
      The `init()` method is async and must be awaited before using any other Renovation Core features.
    </Note>
    
    ### Configuration Options
    
    The `init()` method accepts an `InitParams` object with the following properties:
    
    - `hostURL` (required): Your Frappe backend URL
    - `backend` (optional): Backend type, defaults to `'frappe'`
    - `clientId` (optional): Custom client identifier for HTTP requests
    - `disableLog` (optional): Set to `true` to disable all logging
    - `useJWT` (optional): Enable JWT authentication (requires `renovation_core` backend app)
  </Step>
  
  <Step title="Authenticate a user">
    Log in a user with email and password:
    
    ```typescript
    const loginResponse = await renovation.auth.login({
      email: 'user@example.com',
      password: 'secure-password'
    });
    
    if (loginResponse.success) {
      console.log('Logged in as:', loginResponse.data.full_name);
      console.log('User roles:', loginResponse.data.roles);
    } else {
      console.error('Login failed:', loginResponse.error);
    }
    ```
    
    <Tip>
      All Renovation Core methods return a `RequestResponse` object with `success`, `data`, and `error` properties.
    </Tip>
  </Step>
  
  <Step title="Perform document operations">
    Once authenticated, you can perform CRUD operations on documents:
    
    ### Get a document
    
    ```typescript
    const docResponse = await renovation.model.getDoc({
      doctype: 'User',
      docname: 'user@example.com'
    });
    
    if (docResponse.success) {
      console.log('User data:', docResponse.data);
    }
    ```
    
    ### Get a list of documents
    
    ```typescript
    const listResponse = await renovation.model.getList({
      doctype: 'User',
      fields: ['name', 'email', 'full_name'],
      filters: [['enabled', '=', 1]],
      limitPageLength: 10,
      orderBy: 'creation desc'
    });
    
    if (listResponse.success) {
      console.log('Users:', listResponse.data);
    }
    ```
    
    ### Create a new document
    
    ```typescript
    const newDoc = await renovation.model.newDoc({
      doctype: 'ToDo'
    });
    
    newDoc.description = 'Complete documentation';
    newDoc.priority = 'High';
    
    const saveResponse = await renovation.model.saveDoc({
      doc: newDoc
    });
    
    if (saveResponse.success) {
      console.log('Created ToDo:', saveResponse.data.name);
    }
    ```
    
    ### Update a document
    
    ```typescript
    const todoResponse = await renovation.model.getDoc({
      doctype: 'ToDo',
      docname: 'TODO-0001'
    });
    
    if (todoResponse.success) {
      const todo = todoResponse.data;
      todo.status = 'Closed';
      
      const updateResponse = await renovation.model.saveDoc({
        doc: todo
      });
      
      if (updateResponse.success) {
        console.log('Updated ToDo status');
      }
    }
    ```
    
    ### Delete a document
    
    ```typescript
    const deleteResponse = await renovation.model.deleteDoc({
      doctype: 'ToDo',
      docname: 'TODO-0001'
    });
    
    if (deleteResponse.success) {
      console.log('Document deleted successfully');
    }
    ```
  </Step>
</Steps>

## Complete Example

Here's a complete example putting it all together:

```typescript app.ts
import { Renovation } from '@leam-tech/renovation-core';

async function main() {
  // Initialize Renovation Core
  const renovation = new Renovation();
  
  await renovation.init({
    hostURL: 'https://your-frappe-site.com',
    backend: 'frappe'
  });
  
  // Login
  const loginResponse = await renovation.auth.login({
    email: 'user@example.com',
    password: 'secure-password'
  });
  
  if (!loginResponse.success) {
    console.error('Login failed:', loginResponse.error);
    return;
  }
  
  console.log('Logged in as:', loginResponse.data.full_name);
  
  // Create a new ToDo
  const newTodo = await renovation.model.newDoc({ doctype: 'ToDo' });
  newTodo.description = 'Learn Renovation Core';
  newTodo.priority = 'High';
  
  const saveResponse = await renovation.model.saveDoc({ doc: newTodo });
  
  if (saveResponse.success) {
    console.log('Created ToDo:', saveResponse.data.name);
  }
  
  // Get list of pending todos
  const listResponse = await renovation.model.getList({
    doctype: 'ToDo',
    fields: ['name', 'description', 'status'],
    filters: [['status', '=', 'Open']],
    limitPageLength: 20
  });
  
  if (listResponse.success) {
    console.log('Pending ToDos:', listResponse.data);
  }
  
  // Logout
  await renovation.auth.logout();
  console.log('Logged out successfully');
}

main().catch(console.error);
```

## Understanding Request Responses

All Renovation Core methods return a `RequestResponse<T>` object with the following structure:

```typescript
interface RequestResponse<T> {
  success: boolean;        // Whether the request succeeded
  data: T;                 // Response data (if success is true)
  error?: RenovationError; // Error details (if success is false)
  httpCode: number;        // HTTP status code
  exc?: string;            // Exception message from backend
}
```

Always check the `success` property before accessing `data`:

```typescript
const response = await renovation.model.getDoc({
  doctype: 'User',
  docname: 'user@example.com'
});

if (response.success) {
  // Safe to access response.data
  console.log(response.data);
} else {
  // Handle error
  console.error(response.error.title, response.error.info);
}
```
